## üì± Mobile Reference

Save this for quick access on your phone:

**Quick Start:**
1. Install Node.js
2. Terminal 1: `npm install` then `npm start` (in server/)
3. Terminal 2: `curl -X POST http://localhost:3000/api/admin/seed`
4. Terminal 3: `python -m http.server 5520` (in root)
5. Browser: `http://localhost:5520/coffee.html`

**Key URLs:**
- App: http://localhost:5520/coffee.html
- API: http://localhost:3000/api/menu

**Common Commands:**
- View menu: `curl http://localhost:3000/api/menu`
- Seed DB: `curl -X POST http://localhost:3000/api/admin/seed`
